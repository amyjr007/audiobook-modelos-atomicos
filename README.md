# Audiobook — Modelos Atômicos

Audiobook interativo e narrado sobre a história dos modelos atômicos, voltado ao ensino de Química.

## Sobre

Aplicativo web (HTML/CSS/JavaScript, sem dependências externas além das fontes do Google Fonts) que apresenta uma jornada lúdica pela evolução do conceito de átomo.

### Funcionalidades

- **Capa** animada com identidade visual em tons de laranja.
- **Aprenda** — seção "Introdução à Química" com experiência de audiobook:
  - Narração sincronizada (`audio/dialogo0.mp3`).
  - Sequência de textos cronometrada ao tempo real do áudio.
  - Controle de **play/pausa** que congela e retoma os eventos sem dessincronizar.
  - **Carrossel interativo** de imagens com gesto de swipe.
- Tópicos adicionais: *Os filósofos do invisível* e *John Dalton e o Átomo*.
- **Quiz** (em construção).

## Como usar

Abra o arquivo [`audiobook_modelos_atomicos.html`](audiobook_modelos_atomicos.html) em um navegador moderno.

## Estrutura

```
audiobook_modelos_atomicos.html   # aplicativo completo
audio/                            # narrações
images/                           # imagens do carrossel
```
